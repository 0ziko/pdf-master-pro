FAQ

Frequently Asked Questions

What is a color palette?
A color palette is a carefully chosen set of colors used together in a design project. Good palettes create visual harmony and brand consistency. They are used in websites, apps, logos, presentations, illustrations, and any creative work that needs a cohesive visual identity.
What are color harmonies?
Color harmonies are rules for combining colors in visually pleasing ways, based on their positions on the color wheel. Complementary colors sit directly opposite each other (high contrast). Analogous colors are adjacent (soft, natural look). Triadic colors are equally spaced (vibrant, balanced). Split-complementary uses one color plus two adjacent to its complement. Monochromatic uses a single hue at different lightness levels.
Can I export the palette?
Yes. Click any swatch to copy its HEX code, or use Copy All HEX to copy every color at once. The Export CSS Variables button copies the palette as :root { --color-1: #hex; ... } CSS custom properties — ready to paste directly into your stylesheet.
Is this tool free?
Yes, completely free. No account required, no watermarks, no usage limits. The color palette generator runs entirely in your browser — no data is ever sent to any server. Use it as many times as you like.
What color formats does it support?
Each swatch displays its color in HEX format (e.g. #8250ff) and RGB format (e.g. rgb(130, 80, 255)). Clicking a swatch copies the HEX code. The CSS Variables export also uses HEX values. Internally, the harmony calculations are performed in HSL for accuracy.
