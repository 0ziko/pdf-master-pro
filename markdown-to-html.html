FAQ

Frequently Asked Questions

What is Markdown?
Markdown is a lightweight markup language created by John Gruber in 2004. It lets you write documents using a simple, readable plain-text syntax — for example # for headings or **bold** — that is then converted to HTML. It is widely used in README files, documentation, wikis, and blog posts.
Does it support all Markdown features?
The converter supports CommonMark basics: headings h1–h6, bold, italic, inline code, fenced code blocks, unordered lists, ordered lists, blockquotes, hyperlinks, and horizontal rules. Extended syntax such as tables, task lists, and footnotes is not currently supported.
Is my text uploaded anywhere?
No. Everything runs entirely in your browser using JavaScript. Your text is never sent to any server — it stays completely private on your device.
Can I download the HTML file?
Yes. Click the "Download .html" button and a complete, self-contained HTML file containing your converted content will be saved to your device instantly, ready to open in any browser.
What's the difference between Markdown and HTML?
Markdown is a human-friendly shorthand writing format designed to be easy to read and write. HTML (HyperText Markup Language) is the actual markup language that browsers render. Markdown is converted into HTML so browsers can display it correctly — this tool performs that conversion instantly, directly in your browser.
